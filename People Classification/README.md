# Person Classification

To train the following models for person classification run :
`python3 ModelTrainer.py`

## Requirements
The code was run using: 
1. Python3
2. Tensorflow 2.2
3. PyTorch 1.3.1
4. Pillow 5.0
5. VisualWakeWords (https://github.com/Mxbonn/visualwakewords)
6. Numpy
7. Matplotlib
